# (C# 코딩) SimpleCalculator

## 개요
- C# 프로그래밍 학습을 위한 간단한 계산기 프로그램
- 사용자 입력을 받아 사칙연산을 수행하고 결과를 출력

- 사용한 플랫폼:
C#, .NET Windows Forms, Visual Studio, GitHub

- 사용한 컨트롤:
TextBox, Button, Label, TAbleLayoutPanel

- 사용한 기술과 구현한 기능:
- 이벤트 기반 프로그래밍 (Button Click Event)
- 문자열 → 정수 변환 (int.Parse)
- 조건문을 활용한 사칙연산 처리
- TextBox 2개를 활용한 입력/수식 분리 출력
- C, CE, Del 기능 구현

### 과제내용
- TextBox와 Button을 이용한 입력 UI 구성
- 숫자 버튼 클릭 시 TextBox에 값 출력
- 간단한 사칙연산 구현(더하기)

### 구현내용과 기능설명
- 버튼 클릭 이벤트를 통해 사용자 입력을 처리
- 여기서 추가로 숫자버튼들 하나 하나에 클릭 이벤트를 연결하여 TextBox에 숫자가 입력되도록 구현
- TextBox에 입력된 값을 읽고 출력하는 기본 구조 구현
- txtDisplay: 현재 입력값 및 결과 출력
- txtExpression: 전체 수식 표시
- 버튼 클릭 시 두 TextBox를 동시에 업데이트
- 숫자들을 이용해 처음엔 간단한 더하기만 구현하였습니다.
- 마지막으로 텍스트 박스를 2개를 만들어 하나는 입력창 하나는 결과창으로 나누어 구별하여 구현하였습니다.

### 구현내용과 기능설명
- 사칙연산 버튼 클릭 시 첫 번째 숫자(num1)와 연산자(op)를 저장
- 다음으로 숫자 버튼 클릭 시 두 번째 숫자(num2)를 저장

- 하지만 컴퓨터는 곱하기 나누기 기호를 인식하지 못하고 / *만을 인식하기 때문에
따로 조건문을 추가하여 곱하기 나누기 버튼에는 * / 기호를 저장하도록 구현하였습니다.

### 과제내용
- 계산기의 수정 및 삭제 기능 구현
- C, CE, Del 버튼을 이용한 입력값 관리

### 구현내용과 기능설명

계산기에서 입력된 값을 수정하거나 삭제하는 기능을 구현하였습니다.

1. C 버튼 (Clear)
- 현재까지 입력된 모든 값과 수식을 초기화
- txtDisplay, txtExpression, 저장된 변수(num1, op)를 모두 초기 상태로 되돌림

예시:
12 + 100 → C → (전체 초기화)

---

2. CE 버튼 (Clear Entry)
- 마지막으로 입력한 피연산자(숫자)를 통째로 삭제
- 현재 입력 중인 값(txtDisplay)을 비우고, 수식(txtExpression)에서도 마지막 숫자를 제거

예시:
12 + 100 → CE → 12 +

---

3. Del 버튼 (Delete)
- 마지막으로 입력한 숫자 한 글자만 삭제
- txtDisplay와 txtExpression에서 각각 마지막 문자 제거

예시:
100 → Del → 10

---

### 추가 설명
- Del은 문자 단위 삭제, CE는 숫자 단위 삭제, C는 전체 삭제로 구분하여 구현하였다.
- string.Substring()을 이용하여 문자열을 잘라내는 방식으로 삭제 기능을 구현하였다.
- 사용자의 입력 상황에 따라 적절한 삭제 동작이 이루어지도록 조건문을 활용하였다.
