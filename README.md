# (C# 코딩) SimpleCalculator

## 개요
- C# 프로그래밍 학습을 위한 간단한 계산기 프로그램
- 사용자 입력을 받아 사칙연산을 수행하고 결과를 출력

- 사용한 플랫폼:
C#, .NET Windows Forms, Visual Studio, GitHub

- 사용한 컨트롤:
TextBox, Button, Label, TAbleLayoutPanel

- 사용한 기술과 구현한 기능:
- 이벤트 기반 프로그래밍 (Button Click Event)
- 문자열 → 정수 변환 (int.Parse)
- 조건문을 활용한 사칙연산 처리
- TextBox 2개를 활용한 입력/수식 분리 출력
- C, CE, Del 기능 구현

### 과제내용
- 계산기의 수정 및 삭제 기능 구현
- C, CE, Del 버튼을 이용한 입력값 관리

### 구현내용과 기능설명

계산기에서 입력된 값을 수정하거나 삭제하는 기능을 구현하였습니다.

1. C 버튼 (Clear)
- 현재까지 입력된 모든 값과 수식을 초기화
- txtDisplay, txtExpression, 저장된 변수(num1, op)를 모두 초기 상태로 되돌림

예시:
12 + 100 → C → (전체 초기화)

---

2. CE 버튼 (Clear Entry)
- 마지막으로 입력한 피연산자(숫자)를 통째로 삭제
- 현재 입력 중인 값(txtDisplay)을 비우고, 수식(txtExpression)에서도 마지막 숫자를 제거

예시:
12 + 100 → CE → 12 +

---

3. Del 버튼 (Delete)
- 마지막으로 입력한 숫자 한 글자만 삭제
- txtDisplay와 txtExpression에서 각각 마지막 문자 제거

예시:
100 → Del → 10

---

### 추가 설명
- Del은 문자 단위 삭제, CE는 숫자 단위 삭제, C는 전체 삭제로 구분하여 구현하였다.
- string.Substring()을 이용하여 문자열을 잘라내는 방식으로 삭제 기능을 구현하였다.
- 사용자의 입력 상황에 따라 적절한 삭제 동작이 이루어지도록 조건문을 활용하였다.

## 구현내용과 기능설명 
여기서는 과제4에서 구현한 기능들을 설명하겠습니다.
괄호와 연속계산들이랑 각종 연산자 우선순위 처리 등등이 추가되어서 과제1,2,3에서 구현했던 기능들을 모두 포함하여 구현하였습니다.
이제 어떻게 구현하였는지 말씀드리겠습니다.
괄호 계산은 DataTable.Compute() 메서드를 활용하여 구현하였습니다. 이 메서드는 문자열로 표현된 수식을 계산할 수 있는 기능을 제공합니다. 이를 통해 괄호가 포함된 복잡한 수식도 쉽게 처리할 수 있습니다.
다음으로 연산자 계산은 기존에 만들어 두었던 함수들이 있기 때문에 어렵지 않았습니다.

- 곱셈, 나눗셈이 덧셈, 뺄셈보다 먼저 계산됨
